## MYSQL + PHPMYADMIN Docker development environment

### Author: Denis Mitrofanov

### Usage
Clone the repo. CD into the folder. Type:
`docker-compose up -d`

Open `localhost:8001` and type: 

**Server**: `dbmysql` 
**Username:** `homestead` 
**Password:** `secret`

### Persistance
All data will be saved into `./db` folder.
